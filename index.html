Clean

Nothing but pure python with a light syntax.

from zoolander import *

sheet = Stylesheet()

with sheet.definitions as rule:

    rule("h1",
         height=px(118px),
         margin_top=em(1))

    rule(".tagline",
         font_size=px(26),
         text_align=right)

Variables

    blue = "#3BBFCE"
    margin = px(16)

    rule(".content_navigation",
         border_color=blue,
         margin=margin/2,
         color=blue.replace("CE", "EF"))

    rule(".border",
         margin=margin+3,
         border_color=blue)

Mixins

    table_scaffolding = dict(text_align=center,
                             font_weight=bold,
                             padding=px(2))

    rule("#blind-me-with-science",
         color="#CCCCEE",
         **table_scaffolding)

    def float_left(distance, **props):
        return dict(float=left,
                    margin_left=distance,
                    **props)

    left_table = float_left(px(10), **table_scaffolding)

    rule("#other-data",
         **left_table)

Render

# Returns the CSS, rendered as a string.
sheet.render()

# Render the CSS to a file
sheet.render_to_file("example.css")

Little Extras

As you may have noticed in the code examples above, Zoolander comes with a few batteries included, short-hand variables. These are provided just so that the number of quotes you need to type are minimized. As of right now, these are the ones that are included:

right = "right"
left = "left"
bold = "bold"
italic = "italic"
none = "none"
block = "block"
inline = "inline"

Install

$ git clone git://github.com/fitzgen/zoolander.git
$ cd zoolander/
$ sudo python setup.py install

Then open the example_styles.py file in your favorite editor, run

$ python example_styles.py

and inspect the output.
